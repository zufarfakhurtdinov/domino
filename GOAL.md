# Original Goal

I have an idea to create a website with a simple language-learning game domino.

It should have dominoes, each of them have two parts and each part can be a text, an image or an audio.

Let's discuss an editor/creator interface later.

Let's assume we have a list of pairs, and each pair is a domino. If it is image or audio, it is in the proper format for web.

The idea is to have a table space, and it will have randomized dominoes scattered everywhere. The dominoes can be vertical or horizontal; angled positions are forbidden.

I want to be able to drag-and-drop a domino. If the domino is close to the other pair domino in the proper place, it should be magneted to it. It's better to have some sort of indication whether it is magneted or not. There should be ability to disjoin magneted dominoes.

I also want to have a button to rotate each domino clockwise by 90 degrees.

That's the main idea. What technologies are best for it? If it is possible to test some logic without full graphics or real data, it's a big advantage. This domino game should run in a browser.
